# Detecci-n-de-intrusiones-ML
Proyecto final Analista de Datos Jr

Sistema de detección de intrusiones en redes (NIDS) basado en Machine Learning. El proyecto analiza tráfico de red del dataset UNSW-NB15, aplica técnicas de preprocesamiento y entrena modelos Random Forest y XGBoost para clasificar tráfico normal y malicioso, priorizando la detección de ataques.
